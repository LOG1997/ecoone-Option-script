## ecoone操作脚本
### 目前功能
1. 告警配置：
    点击页面左侧大红色按钮触发。（目前暂时别使用，未实现自定义配置告警）
2. 修改字体样式
    选中字体后，按键盘‘D’触发，第一次触发会要求选择字体颜色和是否加粗字体。
